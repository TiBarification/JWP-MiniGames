# JWP-MultiGames

### Requirements
- SourceMod 1.7.3 and higher
- [Jail-Warden-Pro](https://github.com/TiBarification/Jail-Warden-Pro)
- [SM_Hosties](https://github.com/dataviruset/sm-hosties)

- Unpack the archive.
- Upload files to the server, observing the structure.
